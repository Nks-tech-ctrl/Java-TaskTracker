# Java CLI Task Tracker

A simple multi-user task tracker built using Java and a command-line interface.

## Features

- Multi-user login
- Add tasks
- View tasks
- Mark tasks as done
- Delete tasks
- Search tasks
- CSV file storage for persistence

## Project Structure
src/
├── Main.java
├── Task.java
└── TaskManager.java
tasks.txt


## How to Run

Compile:javac Main.java TaskManager.java Task.java
Run:java Main
